# Decomposição

- Dividir problemas complexos em subproblemas;
- Estratégia:
  - Análise: estudar, examinar, decompor;
  - Síntese: reconstruir, fundir elementos de maneira coerente;
- Ordem de execução: sequencial, paralelo;
- Como decompor:
  - Identificar, coletar dados;
  - Agregar dados;
  - Funcionalidade;
